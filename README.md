# 🧠 AI-Powered Dyslexia Reading Assistant (Chrome Extension)

## 📌 Overview

This project is a Chrome Extension designed to assist users with dyslexia by improving readability and providing audio support. It transforms webpage text into a more accessible format and reads it aloud using Text-to-Speech.

---

## 🚀 Features

* 🔍 Capture selected text from any webpage
* 🔊 Text-to-Speech (reads text aloud)
* ✨ Dyslexia-friendly reading mode

  * Increased letter spacing
  * Improved line spacing
  * Soft background color
* 🔄 Enable / Disable reading mode
* ⏹ Stop speech anytime
* 🧹 Clear selected text

---

## 🛠️ Tech Stack

* **Frontend**: HTML, CSS, JavaScript
* **Browser APIs**: Chrome Extensions API
* **Storage**: chrome.storage.local
* **Text-to-Speech**: Web Speech API

---

## ⚙️ How It Works

1. User selects text on any webpage
2. Extension captures and stores the text
3. Popup displays the selected text
4. User can:

   * Listen to the text
   * Enable reading mode
   * Improve readability

---

## 📷 Demo Workflow

* Select text → Open extension
* Click "Read Text" → Audio output
* Click "Enable Reading Mode" → Visual changes

---

## 🌍 Real-World Impact

This project helps individuals with dyslexia by:

* Reducing reading difficulty
* Improving comprehension
* Making digital content more accessible

---

## 📦 Installation

1. Download or clone this repository
2. Open Chrome and go to:
   chrome://extensions
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select the `extension` folder
